این‌ها اصلاً چی هستند؟

مرحله ۱ از ۸

قبل از نصب، بیا با یک تشبیه ساده بفهمیم هر کدام چه‌کاره‌اند:

🐍
پایتون (Python) — یک «زبان» برای حرف زدن با کامپیوتر است. همان‌طور که با فارسی با دوستت حرف می‌زنی، با پایتون به کامپیوتر می‌گویی چه کاری انجام بدهد.
🟢
Node.js — یک «موتور» است که زبان دیگری به نام جاوااسکریپت۱ را روی کامپیوترت اجرا می‌کند. مثل موتور ماشین که بنزین را به حرکت تبدیل می‌کند.
💙
VS Code — یک «دفترچه‌ی مخصوص کدنویسی» است. جایی که کدهایت را می‌نویسی، رنگی می‌شوند و اشتباه‌هایت را گوشزد می‌کند.
⚡
پاورشل (PowerShell) — «اتاق فرمان» ویندوز است. یک پنجره‌ی ساده که در آن به‌جای کلیک کردن، دستور تایپ می‌کنی و ویندوز اطاعت می‌کند!

📖 پاورقی ۱ — جاوااسکریپت چیست؟ جاوااسکریپت (JavaScript) زبانی است که بیشتر سایت‌های اینترنت با آن ساخته می‌شوند. Node.js کاری می‌کند این زبان بیرون از مرورگر هم کار کند.
💡 نکته چرا با پاورشل نصب می‌کنیم و نه با دانلود از سایت‌ها؟ چون سریع‌تر است، از سایت‌های تقلبی در امان می‌مانی، و همه‌چیز با یک خط دستور انجام می‌شود!

پیش‌نیازها — چی لازم داریم؟

مرحله ۲ از ۸
۱
یک کامپیوتر با ویندوز ۱۰ یا ۱۱ (بیشتر کامپیوترهای امروزی همین‌طورند).
۲
اینترنت وصل — چون برنامه‌ها از اینترنت دانلود می‌شوند.
۳
حدود ۲ گیگابایت فضای خالی روی دیسک (تقریباً همه‌ی کامپیوترها دارند).
۴
کمی حوصله! 😊 هیچ دانش قبلی لازم نیست.
⚠️ مواظب باش در طول نصب ممکن است پنجره‌ای باز شود و بپرسد «آیا اجازه می‌دهی این برنامه تغییراتی ایجاد کند؟»۲ — این طبیعی است؛ روی Yes (بله) کلیک کن. اگر کامپیوتر مالِ خانواده است، بهتر است با اجازه‌ی یک بزرگ‌تر این کار را انجام بدهی.
📖 پاورقی ۲ — این پنجره‌ی سؤال چیست؟ اسمش UAC است؛ نگهبان ویندوز! هر وقت برنامه‌ای بخواهد چیزی نصب کند، اول از تو اجازه می‌گیرد. چون خودمان دستور نصب را داده‌ایم، اجازه دادن اشکالی ندارد.
✅ خودت را امتحان کن: ویندوز داری؟ اینترنت وصل است؟ (یک سایت باز کن ببین بالا می‌آید.) اگر بله، برو مرحله‌ی بعد!
۳

باز کردن پاورشل ⚡

مرحله ۳ از ۸
۱
روی صفحه‌کلید، دکمه‌ی ویندوز را فشار بده (دکمه‌ای با علامت پنجره ⊞، پایینِ سمت چپِ صفحه‌کلید، کنار Alt).
۲
حالا تایپ کن: powershell (لازم نیست جایی کلیک کنی؛ همین که تایپ کنی، ویندوز خودش جست‌وجو می‌کند.)
۳
در نتیجه‌ها، Windows PowerShell را می‌بینی. روی آن یک بار کلیک کن تا باز شود.
۴
یک پنجره‌ی آبی یا مشکی باز می‌شود که داخلش نوشته‌ای شبیه این است و یک مکان‌نمای چشمک‌زن دارد:
PS C:\Users\Ali> _
💡 نکته به‌جای «Ali» اسم کاربری کامپیوترِ خودت را می‌بینی. این خط یعنی: «پاورشل آماده است، دستورت را بگو!»
📖 توضیح — چطور در پاورشل دستور بدهیم؟ خیلی ساده: دستور را تایپ می‌کنی (یا کپی می‌کنی و با راست‌کلیک داخل پنجره، پیست می‌شود) و بعد دکمه‌ی Enter را می‌زنی. همین! Enter یعنی «اجرا کن».
✅ خودت را امتحان کن: پنجره‌ی پاورشل باز است و مکان‌نمای چشمک‌زن را می‌بینی؟ عالی است! این پنجره را تا آخر آموزش باز نگه دار.
۴

بررسی ابزار جادویی winget 📦

مرحله ۴ از ۸

winget۳ مثل یک «پیک فروشگاه» است: اسم برنامه را به او می‌گویی، خودش می‌رود از فروشگاه رسمی مایکروسافت آن را می‌آورد و نصب می‌کند. اول مطمئن شویم این پیک سرِ کارش هست:

۱
در پاورشل این دستور را تایپ کن و Enter بزن:
winget --version
۲
اگر جوابی شبیه این دیدی، همه‌چیز مرتب است:
v1.7.10582
💡 نکته عددها ممکن است روی کامپیوتر تو کمی فرق کنند — مهم نیست! همین که یک شماره نسخه نشان بدهد یعنی winget آماده است.
⚠️ اگر پیغام قرمز داد! اگر نوشت winget is not recognized یعنی پیک ما هنوز استخدام نشده! نگران نباش — برو به بخش 🔧 رفع اشکال، مشکل شماره ۱، درستش کن و برگرد همین‌جا.
📖 پاورقی ۳ — winget مخفف چیست؟ مخفف Windows Package Manager یعنی «مدیر بسته‌های ویندوز». به هر برنامه‌ی قابل نصب، یک «بسته» می‌گویند — مثل بسته‌های پستی!
✅ خودت را امتحان کن: با زدن winget --version یک شماره نسخه دیدی؟ حالا آماده‌ی نصب اولین برنامه هستیم! 🚀
۵

نصب پایتون 🐍

مرحله ۵ از ۸
۱
این دستور را در پاورشل تایپ (یا کپی) کن و Enter بزن:
winget install --id Python.Python.3.12 -e
۲
اگر بار اول است، شاید بپرسد آیا شرایط استفاده را قبول داری (Do you agree to all the source agreements?). حرف Y را تایپ کن و Enter بزن.۴
۳
حالا صبر کن ☕ — نوارهایی از پیشرفت دانلود می‌بینی. وقتی تمام شود، می‌نویسد:
Successfully installed
۴
خیلی مهم: پنجره‌ی پاورشل را ببند و طبق مرحله‌ی ۳ دوباره باز کن.۵
۵
حالا برای اطمینان، این را بزن:
python --version
۶
باید جوابی شبیه این ببینی:
Python 3.12.4
📖 پاورقی ۴ — این سؤال موافقت چیست؟ winget فقط یک بار می‌پرسد که آیا قبول داری برنامه‌ها را از مخزن رسمی‌اش بیاورد. Y یعنی Yes (بله).
📖 پاورقی ۵ — چرا باید پاورشل را ببندیم و دوباره باز کنیم؟ وقتی برنامه‌ای نصب می‌شود، آدرسش در «دفترچه تلفن» ویندوز ثبت می‌شود. پاورشلِ باز، دفترچه‌ی قدیمی را در دست دارد! با بستن و باز کردن، دفترچه‌ی جدید را برمی‌دارد و پایتون را پیدا می‌کند. پرتکرارترین اشتباه مبتدی‌ها فراموش کردن همین کار است!
💡 نکته عدد 3.12 نسخه‌ی پایتون است. اگر روزی نسخه‌ی جدیدتری آمده بود و این دستور کار نکرد، به بخش رفع اشکال (مشکل ۴) نگاه کن.
✅ خودت را امتحان کن: دستور python --version یک شماره نسخه نشان داد؟ تبریک — پایتون نصب شد! 🐍✨ اگر خطای فروشگاه مایکروسافت باز شد، برو رفع اشکال، مشکل ۳.
۶

نصب Node.js 🟢

مرحله ۶ از ۸

حالا که یاد گرفتی، بقیه مثل آب خوردن است — همان کار، فقط با یک اسم دیگر:

۱
در پاورشل بزن:
winget install --id OpenJS.NodeJS.LTS -e
۲
صبر کن تا Successfully installed را ببینی.
۳
پاورشل را ببند و دوباره باز کن (یادت هست چرا؟ پاورقی ۵! 😉).
۴
این دو دستور را یکی‌یکی بزن تا مطمئن شویم:
node --version
npm --version
۵
باید دو شماره ببینی، چیزی شبیه:
v20.15.1
10.7.0
📖 پاورقی ۶ — LTS و npm یعنی چه؟ LTS یعنی «نسخه‌ی پشتیبانی طولانی» — پایدارترین نسخه، مخصوص کسانی که دردسر نمی‌خواهند؛ برای همین ما همین را نصب کردیم. npm هم فروشگاه ابزارهای Node.js است که خودکار همراهش نصب می‌شود — دوتا نصب کردیم به قیمت یکی! 🎁
✅ خودت را امتحان کن: هر دو دستور شماره نسخه دادند؟ Node.js هم آماده است! 🟢
۷

نصب VS Code 💙

مرحله ۷ از ۸
۱
آخرین دستور نصب! در پاورشل بزن:
winget install --id Microsoft.VisualStudioCode -e
۲
صبر کن تا Successfully installed بیاید.
۳
پاورشل را ببند و دوباره باز کن (بار آخر، قول می‌دهم! 😄).
۴
حالا این را بزن تا VS Code باز شود:
code
۵
چند لحظه بعد، پنجره‌ی آبیِ خوش‌گلِ VS Code باز می‌شود — دفترچه‌ی کدنویسی‌ات آماده است!
💡 نکته از این به بعد می‌توانی VS Code را از منوی استارت هم باز کنی: دکمه‌ی ویندوز ⊞ را بزن، تایپ کن code و Enter.
💡 نکته‌ی حرفه‌ای — فارسی‌سازی داخل VS Code، از منوی سمت چپ روی آیکون مربع‌ها (Extensions) کلیک کن، بنویس persian و بسته‌ی زبان فارسی را نصب کن تا منوها فارسی شوند. (اختیاری است!)
✅ خودت را امتحان کن: پنجره‌ی VS Code باز شد؟ سه‌تایی کامل شد! برویم برای آزمایش نهایی 🏁
۸

آزمایش نهایی — اولین کد زندگی‌ات! 🏁

مرحله ۸ از ۸

بیا مطمئن شویم همه‌چیز واقعاً کار می‌کند — و همزمان اولین برنامه‌ات را بنویسی!

۱
در پاورشل بزن:
python -c "print('Salam Donya!')"
۲
پایتون باید جواب بدهد:
Salam Donya!
۳
حالا نوبت Node.js است:
node -e "console.log('Salam az Node!')"
۴
و جوابش:
Salam az Node!
📖 توضیح — الان چه اتفاقی افتاد؟ تو به دو زبان مختلف به کامپیوتر گفتی «یک جمله چاپ کن» و او اطاعت کرد! print در پایتون و console.log در جاوااسکریپت هر دو یعنی «روی صفحه بنویس». تو الان رسماً برنامه‌نویسی کرده‌ای! 👏

🎉 تمام شد! حالا تو بلدی…

  • 📦 با winget هر برنامه‌ای را با یک خط نصب کنی
  • 🐍 پایتون، 🟢 Node.js و 💙 VS Code را نصب و امتحان کنی
  • 👨‍💻 اولین کدت را اجرا کنی!

قدم بعدی؟ در VS Code یک فایل بساز و شروع کن به یادگیری پایتون. دنیا منتظر برنامه‌های توست! 🚀

رفع اشکال — اگر جایی گیر کردی

مشکل ۱: می‌نویسد winget is not recognized
راه حل یعنی winget روی این ویندوز نیست. دکمه‌ی ویندوز ⊞ را بزن، تایپ کن Microsoft Store و بازش کن. در جست‌وجوی فروشگاه بنویس App Installer و آن را نصب یا به‌روزرسانی (Update) کن. بعد پاورشل را ببند، دوباره باز کن و از مرحله‌ی ۴ ادامه بده.
مشکل ۲: بعد از نصب، python یا node یا code را نمی‌شناسد
راه حل ۹۰٪ مواقع فقط یادت رفته پاورشل را ببندی و دوباره باز کنی (پاورقی ۵). اگر باز هم نشد، یک بار کامپیوتر را ری‌استارت کن — دفترچه تلفن ویندوز کاملاً تازه می‌شود.
مشکل ۳: با زدن python فروشگاه مایکروسافت باز می‌شود!
راه حل این یک میان‌برِ فضولِ ویندوز است. دکمه‌ی ویندوز ⊞ را بزن، تایپ کن manage app execution aliases و بازش کن. در فهرست، دو کلید مربوط به python.exe و python3.exe که نامشان App Installer است را خاموش کن. پاورشل را ببند و باز کن؛ حل شد!
مشکل ۴: می‌گوید بسته پیدا نشد (No package found)
راه حل شاید نام دقیق بسته کمی عوض شده باشد. با این دستور خودت جست‌وجو کن: winget search python (یا node یا vscode). در ستون Id نام دقیق را پیدا کن و همان را در دستور نصب بگذار.
مشکل ۵: دانلود وسط راه قطع شد یا خیلی کند است
راه حل اینترنتت را چک کن و همان دستور نصب را دوباره بزن — winget از نو شروع می‌کند و مشکلی پیش نمی‌آید. اگر باز قطع شد، چند دقیقه بعد یا با اینترنت دیگری امتحان کن.
مشکل ۶: پنجره‌ی اجازه (Yes/No) آمد و ترسیدم!
راه حل نترس 😊 این همان نگهبان ویندوز است (پاورقی ۲). چون خودت دستور نصب داده‌ای، Yes را بزن. اگر گزینه‌ی Yes خاموش است، یعنی حساب کاربری‌ات مدیر نیست — از یک بزرگ‌تر بخواه رمز مدیر را وارد کند.
